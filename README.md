# Smart_Room_Monitor_and_Alert_System
This project is an embedded system that monitors room conditions using the MSPM0G3507 LaunchPad. The system measures temperature and ambient light level, compares them against preset thresholds, and reports the room status in real time.

## Dev Notes
- Paste files into a fresh empty_freertos project
- Update configTOTAL_HEAP_SIZE to (12 * 1024)
- Must view UI using UART (BAUD RATE = 115200)

## How to Use
- Use S1 & S2 buttons to navigate menus
- Enter custom values for select fields with keyboard
- Alter recorded temperature by touching device CPU
